# Igra Nim

### Skupinski projekt pri predmetu Umetna inteligenca: spodbujevalno učenje za igro Nim

Primerjava različnih metod spodbujevalnega učenja na tej igri.

### Ideje

* Q-learning
* SARSA algoritem
* izboljšan min-max algoritem (z Alpha Beta Pruning)
